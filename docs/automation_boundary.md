# RepLife Automation Boundary

Allowed on VPS:
- content drafts
- captions
- show outlines
- transcript processing
- social scheduling
- reminders
- analytics summaries
- internal reports
- uptime checks
- cron heartbeats

Not allowed on VPS:
- commerce credentials
- payment webhooks
- store API scripts
- store publishing scripts
- customer/account sync jobs
- order/refund/receipt handling
- fulfillment automation
