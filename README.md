# RepLife Automation Content Operations Package - V4 Step-by-Step

Use this full package with `docs/SETUP_NON_TECHNICAL.md` or the PDF manual `RepLife_Automation_NonTechnical_Setup_Manual_v4_STEP_BY_STEP.pdf`.

This package configures RepLife content operations only. It does not configure or rebuild Task Scheduler App, Financial Command Center, LobeHub, a second n8n, commerce, payments, store operations, customer service, DMs, comment auto-replies, or unattended public posting.

Start with the manual Step 1. Run one command block at a time. Stop if a check fails.
